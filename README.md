# Flowtype::Rails

 A wraper for [flowtype.js](https://github.com/simplefocus/FlowType.JS)

## Installation

Add this line to your application's Gemfile:

    gem 'flowtype-rails'

And then execute:

    $ bundle

And then add the following manifest file (application.js)

    //= require flowtype
